# Roadmap

Gigio Flow is moving toward a public, Codex-first open-source release.

## Phase 1: Open Source Foundation

- [x] Markdown workspace protocol
- [x] squad roles and safety rules
- [x] local Studio foundation
- [x] project knowledge base
- [x] workflow folders for proposals, pending work, and active work
- [ ] public README and contributor docs
- [ ] clean demo workspace
- [ ] screenshots or short demo video

## Phase 2: Codex-First Product Pipeline

- [ ] idea to PRD workflow
- [ ] PRD to small issue refinement
- [ ] LLM-based complexity estimation
- [ ] QA review artifact generation
- [ ] human approval gate
- [ ] knowledge update automation

## Phase 3: Delivery Bridge

- [ ] Linear issue creation
- [ ] Linear webhook sync
- [ ] GitHub branch and PR handoff
- [ ] PR description generation
- [ ] Vercel preview status sync
- [ ] release note and history updates

## Phase 4: Maintainer Experience

- [ ] example projects
- [ ] plugin-ready squad packs
- [ ] stricter security audit workflow
- [ ] GitHub Actions for lint/build checks
- [ ] Codex maintainer recipes

## Phase 5: Ecosystem

- [ ] hosted sync option
- [ ] marketplace for templates and squad packs
- [ ] integrations with GitHub Issues, Jira, and other tools
- [ ] team collaboration model
