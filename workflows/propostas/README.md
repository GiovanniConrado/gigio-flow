# Proposals Folder (PRDs)

This folder gathers the product requirements specifications (PRDs) that were written by the **PM Agent** and the **CTO Agent** using the `.ai/templates/prd.md` template and are awaiting approval from the founder.

- **Flow:** The founder reads the PRD and replies in the chat with the message **"Approved"** or **"OK"**.
- **Post-OK Action:** The AI moves this file to the `workflows/pendentes/` folder (or synchronizes the generated issues in Linear/GitHub) to authorize the start of development.
