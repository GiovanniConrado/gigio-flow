# Pasta de Propostas (PRDs)

Esta pasta reúne as especificações de requisitos de produto (PRDs) que foram redigidas pelo **PM Agent** e pelo **CTO Agent** utilizando o template `.ai/templates/prd.md` e estão aguardando a aprovação do fundador.

- **Fluxo:** O fundador lê a PRD e responde no chat com a mensagem **"Aprovado"** ou **"OK"**.
- **Ação pós-OK:** A IA move este arquivo para a pasta `workflows/pendentes/` (ou sincroniza as issues geradas no Linear/GitHub) para autorizar o início do desenvolvimento.
